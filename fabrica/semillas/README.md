# Semillas

Aqui nacen los mundos nativos antes de convertirse en grafo o JSON.

Una semilla no es un mundo. Es una promesa jugable pequena y comprobable.

Las semillas nuevas usan version 1.2. El Pregenerador traduce elecciones
expresadas en lenguaje normal a `experiencia_jugable`; el usuario confirma una
frase comprensible y no necesita ver ni conocer la taxonomia interna.

Regla: si la semilla no se entiende, no se pasa a arquitectura.

Regla: si genero, tono y premisa estan claros pero no se sabe que significan un
objeto, una deduccion, una decision o un error, la semilla aun no esta cerrada.
