# Visual assets and provenance

The ULLDE:VERSE dossier uses generated raster assets connected to the world through a versioned visual manifest.

## Public asset policy

- Character portraits are completely fictional AI-generated designs.
- No published portrait was generated from a personal photograph.
- Environmental scenes and objects are generated illustrations for the historical fiction.
- Raw prompts, private working files and reference photography are not included.
- The manifest publishes only the final WebP files required by the application.

## Traceability

`fabrica/media/aventura_el_testigo_de_ulldecona_001/visual_manifest_source_v1.json` records:

- stable asset identifier;
- public file;
- semantic function;
- linked nodes or entities;
- approval status;
- source signature.

`public/media/aventura_el_testigo_de_ulldecona_001/manifest.json` is the stripped runtime representation. It contains no provider data, private notes, prompt paths or embedded Base64.

The public character portraits were regenerated on 18 July 2026 using the built-in OpenAI image generation workflow. The final designs do not preserve the identity of any person.
